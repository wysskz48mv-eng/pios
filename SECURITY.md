# Security Policy — VeritasIQ Technologies Ltd

## Supported Versions

All production deployments are maintained at the latest version.
We do not support or patch prior versions.

## Reporting a Vulnerability

VeritasIQ Technologies Ltd takes security vulnerabilities seriously.

**Please DO NOT report security vulnerabilities via GitHub Issues.**

Instead, email: **security@veritasiq.co.uk**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Your contact details (if you wish to be credited)

We will acknowledge receipt within **48 hours** and aim to resolve
critical vulnerabilities within **7 days**.

## Our Security Standards

This platform is certified under:
- Cyber Essentials Plus (UK Government baseline)
- ISO 27001:2022 (in progress — target August 2026)

## Responsible Disclosure

We ask that you:
- Give us reasonable time to fix the issue before any public disclosure
- Do not access, modify, or delete data beyond what is needed to demonstrate the vulnerability
- Do not conduct denial-of-service attacks

We will not take legal action against researchers acting in good faith.
