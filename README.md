# PIOS — Personal Intelligent Operating System

Your AI-powered command centre for academic, consulting, projects, and business operations.

## Stack
- **Frontend:** Next.js 14 / TypeScript / Tailwind CSS
- **Database:** Supabase (PostgreSQL + Auth + RLS)
- **AI:** Claude claude-sonnet-4 (Anthropic)
- **Billing:** Stripe
- **Email:** Resend
- **Deploy:** Vercel

## Plans
- **Student** — $9/mo — Academic Lifecycle + Calendar
- **Individual** — $19/mo — Full MVP (email, projects, expenses)
- **Professional** — $39/mo — FM Consulting + Business Ops

## Quick start
```bash
npm install
cp .env.example .env.local
# Fill in env vars
npm run dev
```
